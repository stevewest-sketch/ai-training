The Capstone Project

Real Impact

The agent you build in this module will be used by real teams at Gladly. This is your opportunity to contribute directly to the AI transformation initiative.

Agent Development Checklist

Follow these steps to build a production-ready agent

Production Agent Checklist
1

Write SPEC.md using planning process

Apply planning mode to define clear requirements, scope, and success criteria.

2

Define inputs, outputs, and integrations

Document what data the agent needs, what it produces, and what systems it connects to.

3

Implement guardrails and approval workflows

Add human-in-the-loop controls for sensitive actions and clear boundaries.

4

Build in sagansystems GitHub org

Set up the repository following Gladly's standards and conventions.

5

Test locally with sample data

Validate functionality, error handling, and edge cases before deployment.

6

Document usage and limitations

Create clear documentation for users including capabilities, limitations, and examples.

7

Create user training materials

Build guides or videos to help users understand and adopt the agent.

8

Deploy and monitor

Launch the agent and set up monitoring for performance and issues.
