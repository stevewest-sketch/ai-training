Key Concepts Covered

Topics you'll learn in this official course

Core Tools

File manipulation, command execution, code analysis, and working with your codebase.

Context Management

/init command, CLAUDE.md files, @ mentions, and how to give Claude the right context.

Hotkeys & Commands

Keyboard shortcuts and slash commands for efficient conversation flow.

Plan & Thinking Mode

Using planning mode for complex tasks and thinking mode for detailed reasoning.

Why This Course Matters

This course teaches the official, Anthropic-recommended ways to use Claude Code. Learning these patterns early will make everything else in this path much easier.
