The CTFC Framework

Structure your prompts with these four components for consistently better results

prompt-framework.md
CONTEXT: [Who you are and what you're working on]

TASK: [Specific action you need]

FORMAT: [How you want the output structured]

CONSTRAINTS: [Any limitations or requirements]
Why This Works

Giving Claude context helps it understand your perspective. Being specific about the task eliminates guesswork. Defining the format ensures you get usable output. Adding constraints prevents unwanted content.

Key Prompting Principles

Additional tips for getting the best results

Be Specific About Output Format

If you need a list, say "bullet points." If you need a table, describe the columns. If you need code, specify the language. Don't make Claude guess.

Provide Examples When Helpful

If you want a specific style or format, show Claude an example. "Write in a tone like this: [example]" is more effective than trying to describe the tone.

Iterate Based on Results

If the first response isn't quite right, tell Claude specifically what to change. "Make it more concise" or "Focus more on the technical details" helps Claude adjust.

The Styles Feature

Claude has built-in styles that control the tone and format of responses

Style What It Does Best For
Normal Balanced, conversational General use
Concise Brief, direct Quick answers
Explanatory Detailed with context Learning, complex topics
Formal Professional tone External communications

Creating Custom Styles

Create your own styles for consistent outputs across conversations

  1. Go to Settings ‚Üí Styles
  2. Click Create New Style
  3. Write instructions for tone, format, approach
Example: "Executive Brief" Style
Write for executive audiences:

- Lead with the insight or recommendation
- Max 3 paragraphs
- Use numbers and metrics when available
- End with clear next steps
When to Use Custom Styles

Create custom styles for recurring communication patterns: team updates, customer emails, executive summaries, technical documentation. Once created, select the style from the dropdown to apply it automatically.
