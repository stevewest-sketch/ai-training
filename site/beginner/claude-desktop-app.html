The Claude Product Family

Claude has three distinct products. This path focuses on Claude Chat.

Product What It's For Where You'll Learn It
Claude Chat Conversational AI for questions, brainstorming, content creation This path (Beginner)
Claude Cowork File-based collaboration, planning, deeper automation Intermediate path
Claude Code Terminal-based development, building agents Advanced path
Focus for This Path

This Beginner path focuses on Claude Chat. The other modes are covered in later paths. In the next module, you'll dive deep into everything Chat can do.

Choosing the Right Mode

Quick reference for when to use each Claude mode

I Want To... Use This Why
Ask a quick question Chat Fastest for Q&A
Draft an email or document Chat Conversational iteration
Brainstorm ideas Chat Back-and-forth works well
Organize files on my computer Cowork Needs file system access
Analyze local spreadsheets Cowork Works with files directly
Have Claude work while I do other tasks Cowork Runs independently
Build an application Code Development environment
Work with Git/version control Code Code-level operations
Quick Decision Guide

1. Quick question or conversation? ‚Üí Chat
2. Need to access/modify local files? ‚Üí Cowork
3. Building something with code? ‚Üí Code
