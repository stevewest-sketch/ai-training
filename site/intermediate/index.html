Building with Claude

Create Claude Skills and Projects to automate your own workflows

Who should take this?

This path is for anyone contributing to agent development or wanting to automate their work. You should have completed the Beginner Path first to ensure you have the foundational knowledge needed.

What You'll Gain

After completing this path, you'll be able to use Claude Code's full capabilities, apply planning mode to reduce iteration cycles by 50%+, create reusable Claude Skills, and build your first working automations.
