INTERMEDIATE PATH ASSESSMENT

Path Assessment

Demonstrate your Claude Cowork skills through self-reflection and practical work

When to Take This Assessment

Complete this assessment after finishing all 6 modules in the Intermediate path. You'll reflect on your knowledge and complete 2 practical demonstrations.

Part 1: Knowledge Self-Assessment

Rate your confidence on each skill (1 = not confident, 5 = very confident)

  • 1

    I understand the difference between Claude Chat and Claude Cowork

    I can explain when to use file-based collaboration vs. conversational prompting

  • 2

    I can use Planning Mode effectively

    I know how to voice dump, create a SPEC, go through the interview process, and review plans

  • 3

    I understand Claude Skills structure and creation

    I can explain what a skill.md file contains and how descriptions determine when skills fire

  • 4

    I can design a Project as an agent

    I understand how Projects provide persistent context and can function like specialized agents

  • 5

    I understand Git concepts for Claude Code

    I can explain commits, branches, push/pull, and ask Claude to perform these operations

  • 6

    I can identify automation candidates in my workflow

    I know how to spot repetitive tasks and evaluate them for Claude automation

  • 7

    I have built my Personal OS

    I can set up CLAUDE.md, context files, and MCP connections to generate daily briefings

Self-Assessment Reflection

If you rated any skill below 3, review that module before proceeding. The practical demonstrations will test these skills directly.

Part 2: Practical Demonstrations

Complete any 2 of the following 5 tasks to demonstrate your skills

Task A Custom Skill Creation

  • Identify a repetitive task in your workflow
  • Create a skill.md with description and instructions
  • Test the skill with at least 2 different inputs
  • Iterate on the description until it triggers correctly

Task B Project Agent Design

  • Create a new Project for a specific use case
  • Add relevant context files (instructions, examples, data)
  • Test the Project with real tasks
  • Document what makes this Project effective

Task C Documented Automation Candidate

  • Identify a process you do weekly or more often
  • Document the current manual steps
  • Write a SPEC for automating it with Claude
  • Identify inputs, outputs, and potential edge cases

Task D Personal OS Setup

  • Create your CLAUDE.md brain file with your role context
  • Set up at least 2 MCP connections (Slack, Gmail, Calendar, or Drive)
  • Run "Let's start our day" and get a daily briefing
  • Customize one aspect based on what's missing or too verbose

Task E File-Based Cowork Task

  • Upload a real work file to Claude Cowork
  • Complete a meaningful transformation or analysis
  • Use @ mentions to reference specific files
  • Export or save the results for actual use
What Counts as Complete

For each task, you should have something tangible to show: a skill.md file, a Project you can demo, a documented SPEC, or output from a Cowork session. The goal is to prove you can apply these concepts to real work.

Completion Criteria

You've successfully completed the Intermediate path when:

  • ✓

    All self-assessment ratings are 3 or higher

    You feel confident in the core concepts covered in each module

  • ✓

    You've completed 2 practical demonstrations

    You have tangible artifacts showing applied skills

  • ✓

    You're using Claude Cowork in your daily work

    File-based collaboration has become part of your workflow
